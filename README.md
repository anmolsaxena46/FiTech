# FiTech
---

FiTech is a modern, responsive fitness web application designed to showcase different stakeholder plans, subscription programs, and career opportunities. It combines a sleek UI with gradient backgrounds, interactive cards, and a fixed navigation header for a professional look.


##  Features
- **Responsive Design**: Works across desktop, tablet, and mobile devices.
- **Fixed Header Navigation**: Easy access to Home, Gym from Home, and Signup pages.
- **Hero Section**: Eye-catching background image with animated zoom effect.
- **Subscription Plans**: Cards for Elite, Pro, and Select customers with dedicated links.
- **Additional Services**: Transform & Coach, Workout programs.
- **Join the Family**: Career and Franchise opportunities highlighted.
- **Footer Section**: Contact details, social links, and legal information.


##  Technologies Used
- **HTML5** for structure
- **CSS3** for styling and animations
- **Google Fonts (Montserrat)** for typography
- **Unsplash** for hero background image
